# Expired Sales Orders 

## Description

This module provides visibility over Sales Orders whose scheduled delivery date has expired and are not fully invoiced.

It allows users to identify sales orders that:
- Have a Scheduled Delivery Date earlier than today.
- Are not 100% invoiced.

The module improves financial and operational control by helping detect pending invoicing situations in overdue orders.

## Usage

After installing the module:

1. Navigate to:
   **Sales Management → Transactions → Expired Sales Orders**
2. Review the list of sales orders that meet the criteria.
3. Use filtering and sorting options to analyze pending cases.

The window is read-only and intended for monitoring and control purposes.
